# SuperImg Examples

## Templates (CLI Workflow)
Use with `superimg init`, `superimg dev`, `superimg render`. See `_templates.json` for the full index.

- **hello-world** — Basic animated title card
- **countdown** — Countdown timer animation
- **stats-card** — Animated stat with counting number and progress bar
- **lower-thirds** — Broadcast-style lower third graphic
- **compose-demo** — Multi-scene composition with transitions
- **watermark-demo** — Overlay watermark via `config.watermark`
- **svg-draw / svg-morph / svg-shapes / svg-text-path / svg-filter / svg-reveal** — Vector motion and SVG effect primitives
- …50+ more across `basics`, `marketing`, `events`, `social`, `interfaces`, `data`, `vector`, and `developer`

## Code-First (SDK)
Use the SuperImg SDK programmatically.

- **typescript** — TypeScript template files

## Apps
Interactive browser applications.

- **hover-preview** — Hover to preview frames
- **player-vite** — Video player with Vite
- **react-player** — React video player component
