# Assets Demo

Template examples demonstrating how to work with assets in SuperImg.

## Features

- **Backgrounds**: Learn how to incorporate background images.
- **Template Defaults**: Default values for asset data merged with scene data.

## Getting Started

1. **Install Dependencies**:
   ```bash
   pnpm install
   ```

2. **Setup**:
   Run the setup script to prepare the environment:
   ```bash
   pnpm setup
   ```

3. **Preview a template**:
   ```bash
   pnpm exec superimg dev templates/intro.ts
   ```

## Structure

This example showcases template patterns with defaults and asset usage. Explore the `templates/` directory to see the implementation details.
