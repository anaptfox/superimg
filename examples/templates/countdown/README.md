# Countdown Timer Example

Animated countdown (5 → 0 → GO!) with pulsing numbers and particle effects.

## Usage

```bash
# Development mode (browser preview)
pnpm dev

# Render to MP4
pnpm render
```

## Features

- Countdown from 5 to 0 with "GO!" finale
- Pulsing number animation on each tick
- Circular progress ring
- Floating particle effects
- Portrait format optimized for social media

## Configuration

- **Resolution**: 1080x1920 (portrait)
- **Duration**: 5 seconds
- **FPS**: 30
